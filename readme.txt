This is a Solidity (Ethereum Blockchain) Remix IDE project.
1) Copy and paste the code in REMIX IDE, for 2 different contracts.
2) Deploy it.

This is a shared Wallet, wherin the owner can deposit any amount of money (Ether) into the smart contract.
The owner can add users here, users can be workers, child etc who are assigned their own address.
Their corresponding limits are set by the owner. The users other than owner can transact within their limits.
There are safety feature added here such that, owner protection is served.

This can be deployed in an environment where Etheruem is a standard way of transaction.
