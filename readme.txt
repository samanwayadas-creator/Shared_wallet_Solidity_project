This is a Solidity Remix IDE project.
1) Copy and paste the code in REMIX IDE, for 2 different contracts.
2) Deploy it.

This is a shared Wallet, wherin the owner can deposit any amount of money into the smart contract,
The ownder can add users here, and add there limits. The other users can transact within their, limits
among different accounts. There are safety feature added here such that, owner protection is served.

This can be deployed in an environment where Etheruem is a standard way of transaction.
